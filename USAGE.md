## 快速开始

### 首次访问
1. 打开应用URL：`https://yourusername.github.io/ielts3`
2. 点击 **"使用 Google 登录"** 按钮

## 清单模块

### 基本功能

#### 添加任务
1. 在顶部输入框输入任务内容
2. 点击 **"添加"** 按钮或按 **Enter** 键
3. 任务立即添加到列表
4. 自动同步到云端

```
例如：输入 "完成雅思练习" → 点击添加
```

#### 完成任务
- **方法1**：点击任务左侧的复选框
- **方法2**：点击任务文字区域
- 已完成的任务会显示删除线样式
- "今日完成"数字自动更新

#### 删除任务
1. 点击任务右侧的垃圾桶图标
2. 任务立即删除
3. 自动从云端同步删除

### 数据管理

#### 同步状态
- **绿色点** = 已连接，数据正在同步
- **无点显示** = 未登录或未连接

#### 数据备份
- 所有数据自动保存在Firebase云端
- 不同设备登录同一账号可看到相同任务
- 删除任务后无法恢复（建议谨慎操作）

#### 离线使用
- 离线状态下仍可添加/编辑任务
- 任务保存在设备本地
- 联网后自动同步到云端

## IELTS分析模块

### 模块概览

IELTS模块分为两个主要部分：
1. **学情录入** - 输入练习数据
2. **趋势分析** - 查看成绩统计

### 学情录入

#### 第一步：选择日期
1. 点击日期输入框（默认为今日）
2. 选择要记录的日期
3. 页面自动加载该日期的数据（如存在）

#### 第二步：输入听力数据
每个Part需要输入：
- **题材**：从下拉菜单选择
  - 例：租房/房产咨询、旅游咨询等
- **得分**：输入正确题数（0-10）

```
Part 1: 租房/房产咨询 - 8 题正确
Part 2: 旅游咨询 - 9 题正确
Part 3: 购物/生活投诉 - 7 题正确
Part 4: 银行开户 - 8 题正确
```

总计：32题正确 → 预估Band 7.5

#### 第三步：输入阅读数据
每个Passage需要输入：
- **题材**：从下拉菜单选择
- **得分**：正确题数（0-13）
- **用时**：花费时间（分钟）

```
Passage 1: 自然科学 - 11 题 - 18分钟
Passage 2: 气候变化 - 12 题 - 19分钟
Passage 3: 能源科技 - 10 题 - 20分钟
```

总计：33题正确，57分钟 → 预估Band 8.0

#### 第四步：记录学习笔记
在下方大文本框记录：
- 今天的学习重点
- 出现的生词
- 错题复盘
- 改进方向

**提示**：笔记会在"笔记导出"时包含

#### 自动保存
- 数据每30秒自动保存一次
- 也可手动刷新页面保存

### 趋势分析

#### 查看总体成绩
页面顶部显示三个指标：
- **听力预估均分** - 基于历史平均得分
- **阅读预估均分** - 基于历史平均得分  
- **总分参考** - 两个科目均分的平均值

```
例如：
听力：7.5 Band
阅读：8.0 Band
总分：7.75 Band (总正确65题/80)
```

#### 对题趋势图表
1. **折线图展示**：听力、阅读、总分的历史趋势
2. **交互功能**：
   - 鼠标悬停 = 显示该日期具体数据
   - 滚动缩放 = 放大/缩小查看细节
   - 拖动 = 左右移动日期
3. **图例点击**：隐藏/显示对应科目的曲线

#### 阅读用时监控
- 显示阅读部分的总用时趋势
- 帮助优化答题速度
- 目标：趋势下降（答题速度加快）

#### 题材弱项云图
两部分展示：
- **听力错题分布** - 哪些题材容易出错
- **阅读错题分布** - 哪些题材容易出错

**云图说明**：
- 字体越大 = 该题材错误越多
- 颜色越深 = 优先级越高
- 帮助针对性复习

### 数据导出

#### Excel导出
1. 点击 **"Excel 导出"** 按钮
2. 下载 IELTS_Full_Report_日期.xlsx 文件
3. 在Excel中查看：
   - 所有日期的完整数据
   - 题材和得分信息
   - 学习笔记

#### 笔记导出
1. 点击 **"笔记导出"** 按钮
2. 下载 IELTS_Notes_日期.txt 文件
3. 在文本编辑器中查看：
   - 按日期组织的笔记
   - 便于回顾和总结

## 常见问题

### 数据相关

**Q：我的数据会丢失吗？**
A：不会。数据存储在Firebase云端，并有备份。只要不主动删除，数据永久保存。

**Q：如何在多设备间同步？**
A：使用同一Google账号登录，数据自动同步。支持电脑、手机、平板等。

**Q：删除的任务能恢复吗？**
A：不能。建议删除前确认。如需找回，可联系开发者尝试从备份恢复。

**Q：离线时能使用吗？**
A：可以。但需要手动刷新页面前已登录过一次。离线数据在联网后自动同步。

### 功能相关

**Q：怎样修改已添加的任务？**
A：目前不支持编辑，需要删除后重新添加。（后续版本可能支持）

**Q：IELTS模块的"预估Band"准确吗？**
A：这是基于题数的粗略估算。实际成绩还取决于难度、其他科目等因素。建议用于学习进度追踪。

**Q：如何导入之前的数据？**
A：可以将Excel或CSV转换为模块要求的格式，再通过导入功能加入。

**Q：可以删除某一天的所有数据吗？**
A：目前没有批量删除功能。需要手动删除或联系开发者。

### 技术问题

**Q：登录失败怎么办？**
A：
1. 检查网络连接
2. 尝试清除浏览器缓存（Ctrl+Shift+Delete）
3. 使用其他浏览器重试
4. 检查Google账号是否正常

**Q：页面打不开？**
A：
1. 确认URL正确：https://yourusername.github.io/ielts3
2. 等待GitHub Pages部署完成（首次2-3分钟）
3. 尝试刷新页面（Ctrl+F5）
4. 检查网络连接

**Q：数据无法保存？**
A：
1. 检查网络连接
2. 确认登录状态（顶部应显示用户邮箱）
3. 查看浏览器开发者工具（F12）的Console错误
4. 尝试重新登录

### 账号相关

**Q：如何更换Google账号？**
A：点击"退出登录"，再用新账号登录即可。

**Q：一个Google账号的数据与另一个账号隔离吗？**
A：是的。每个Google账号有独立的数据空间。

**Q：如何申请删除账号和所有数据？**
A：联系开发者，需验证身份后执行删除操作。
